特性概览

Antigravity CLI 将 Antigravity 2.0 的核心能力直接带入您的终端。它针对键盘驱动的工作流和远程 SSH 会话进行了优化，资源开销极小。

  • 主要关注点: 速度、键盘效率、低开销。
  • 界面: 终端用户界面 (TUI)。
  • 工作流: SSH/远程会话、键盘优先。
  • 代理引擎: 与 Antigravity 2.0 共享核心代理引擎。

集成

共享代理引擎

两个界面都运行在同一个核心代理引擎上。任何改进都会自动应用于两者。

共享设置

首选项和权限双向同步。

会话导出

将终端会话导出到 Antigravity 2.0 GUI 中继续工作。

安装

macOS / Linux

Bash 
curl -fsSL https://antigravity.google/cli/install.sh | bash

Windows PowerShell

PowerShell 
irm https://antigravity.google/cli/install.ps1 | iex

Windows CMD

CMD 
curl -fsSL https://antigravity.google/cli/install.cmd -o install.cmd && install.cmd && del install.cmd

认证

CLI 通过系统密钥环进行认证，如果没有活动会话，则回退到 Google 登录。

本地

远程 / SSH
